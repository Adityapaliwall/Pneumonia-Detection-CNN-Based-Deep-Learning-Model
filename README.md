## üìò Project Description

Pneumonia is a serious lung infection, and early detection is critical. Manual examination of X-rays can be time-consuming and prone to human error.  
This deep learning project provides an automated solution by:

- Preprocessing chest X-ray images  
- Training a CNN model to extract visual patterns  
- Classifying images as **Normal** or **Pneumonia**  

This model serves as a foundation for future AI-based diagnostic tools in healthcare.
